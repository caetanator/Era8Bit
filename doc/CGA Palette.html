Notes:  
  • Color number 6 Brown (#AA5500) should be Dark Yellow (#AAAA00), but IBM changed the hardware on its CGA monitors to swap the colors (the brown is more useful than the dark yellow). Most of the clones did the same.
  • By default, in text mode, the intensity bit (I), can't be used for the text background color because it is used as the flash bit (F). So it's normally used for the text foreground color.
    This can be changed by altering a CGA register, that changes the Flash bit to Intensity bit, for the background color.
    On graphics mode, this issue doesn't exist, because it's used 4 bit (IRGB) per pixel (2 pixels for each byte) on Hi-Color mode, and 2 bits (palette index) per pixel (4 pixels for each byte) on Hi-Resolution mode. It can also use 1 bit per pixel (8 pixels per byte) on monochrome mode.
  • The 1-byte color attribute for each character, has the following arrangement:
