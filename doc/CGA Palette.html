Notes:  
  • Color number 6 Brown (#AA5500) should be Dark Yellow (#AAAA00), but IBM changed the hardware on it's CGA monitors to swap the colors (the brown is more ussefull than the dark yellow). Most of the clones did the same.
  • By default, in text mode, the intensity bit (I), can't be used for the text background color because it is used as the flash bit (F). So it's normaly used for the text forground color.
    This can be changed by altering a CGA register, that changes the Flash bit to Intensity bit, for the background color.
    On graphics mode, this isue doesn't exist.
  • The one byte color attribute for each character, has the following arrangement:
      Background Forground
    F/I Rp2 Gp1 Bp0 I Ri2 Gi1 Bi0
