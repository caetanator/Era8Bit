Notes:  
  • Color number 8 Black (#00000) is displayed by some clones (specialy on NTSC computers) as Dark Gray (#AAAAAA).
  • On Sinclair ZX Spectrum +128K (Toastrack) and Sinclair ZX Spectrum +2 (Gray), if connected to a CGA monitor by the RGBI conector, the displayed colors are the ones for the CGA standard palette, being the most significant changes:
        - color number 8 Black (#00000) is displayed as as Dark Gray (#AAAAAA);
        - color number 6 Yellow (#AAAA00) is displayed as Brown (#AA5500);
  • Colors are order by GRBI, because that way they are order from the dark colors to the bright colors and that simplifies the conversion to gray scale on black-and-white TVs and monochrome monitores (green, amber and blue phosfer) and improves image quality;
  • The intensity bit (I), can't be used for the 8x8 pixel matrix background color, because it is used as the flash bit (F), by the ZX Spectrum ULA and Timex SCLD.
    Also, it's not used for the border color. It only afects the 8x8 pixel matrix forground/text color.
  • The one byte color attribute for each character (8x8 pixels), has the following arrangement:
